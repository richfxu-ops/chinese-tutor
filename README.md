# HSK-5 中文 Tutor

Fine-tune **Qwen2.5-7B-Instruct** into a patient HSK-5-level Mandarin tutor with **QLoRA**, on **synthetic data generated by Claude**, and serve it as a **Gradio chat demo** with a bilingual + pinyin + hover-gloss reading layer.

> A side project for modern-fine-tuning-stack fluency and a genuinely useful study tool. Small and shippable, not research-grade.

## What it does

Chat in Chinese (or English when stuck). The tutor answers at HSK-5 level across **six tasks**:

| Task | You ask | It does |
|---|---|---|
| 解词 explain a word | `"毕竟"是什么意思？` | meaning, usage, an example |
| 造句 use in a sentence | `用"承担"造两个句子` | natural example sentences |
| 改错 correct my Chinese | `我昨天去过北京。` | problem → fix → one-line why |
| 翻译 translate | `How do I say "I can't help but worry"?` | translation, HSK-5 level |
| 对话 example dialogue | `给我一段点菜的对话` | a short natural dialogue |
| 语法 grammar point | `怎么用"既然……就……"？` | structure + examples |

Every answer is **fully bilingual**. The interface layers **pinyin over each character** and a **hover gloss on each word** — done deterministically app-side (`pypinyin` + `jieba` + CC-CEDICT), not by the model, so pinyin is always correct.

## How it's built

```
seed lists ──▶ gen_data.py ──▶ train.py ──▶ eval.py ──▶ merge.py ──▶ GGUF ──▶ app.py
 (HSK-5)        (Claude)       (QLoRA,       (before/    (adapter    (quantize   (Gradio +
                               Colab)        after)      → fp16)     Q4_K_M)     reading layer)
```

- **Data** is generated locally with Claude; **training/merge/quantize** run on Colab (bitsandbytes 4-bit is CUDA-only); the **demo** runs locally on Apple Silicon via llama.cpp (Metal).
- One `config.py` is the source of truth for every stage.

## Quickstart

### 1. Generate data (local, needs an Anthropic key)
```bash
pip install -r requirements.txt
export ANTHROPIC_API_KEY=sk-ant-...
python gen_data.py --dry-run     # free: inspect the prompts
python gen_data.py --limit 3     # cents: eyeball real output
python gen_data.py               # full ~900 → data/train.jsonl, data/eval.jsonl
```

### 2. Train + package (Colab GPU)
Open [`train_colab.ipynb`](train_colab.ipynb) in Colab (set runtime to a GPU), upload `config.py` / `train.py` / `eval.py` / `merge.py` / the two `.jsonl`, and run the cells: sanity run → full train → eval → merge → GGUF quantize → download `hsk5-tutor-q4_k_m.gguf`. Put it in `outputs/`.

### 3. Run the demo (local, Apple Silicon)
```bash
python get_cedict.py             # once: fetch CC-CEDICT for hover glosses
python get_strokes.py            # once: fetch stroke data for 写 practice (~46MB)
python app.py                    # serves the GGUF at http://localhost:7860
```

Or double-click `start-tutor.command` (launches and opens the browser for you).

`LAN=1 python app.py` also serves on your Wi-Fi (the printed URL) so a phone or
iPad can be the touchscreen — nice for 写 stroke practice. Localhost-only by default.

## Repo layout

| File | Role |
|---|---|
| `config.py` | single source of truth (models, paths, tutor `SYSTEM_PROMPT`, task specs, QLoRA hyperparams) |
| `hsk5_vocab.txt` · `hsk5_grammar.txt` | curated HSK-5 seed the generator draws from |
| `gen_data.py` | Claude teacher → `data/*.jsonl` |
| `train.py` · `train_colab.ipynb` | QLoRA SFT + the Colab pipeline |
| `eval.py` | base vs. tuned before/after + optional Claude-judge rubric |
| `merge.py` | fold the LoRA adapter into an fp16 model |
| `annotate.py` · `get_cedict.py` | the reading layer (pinyin ruby + hover gloss) |
| `app.py` | Gradio chat demo |
| `planning/` | plan, architecture, decisions, task board + dashboard |

## Notes & attribution

- **Base model:** [Qwen2.5-7B-Instruct](https://huggingface.co/Qwen/Qwen2.5-7B-Instruct) (Apache-2.0).
- **Dictionary:** [CC-CEDICT](https://www.mdbg.net/chinese/dictionary?page=cc-cedict), CC-BY-SA 4.0 — fetched by `get_cedict.py`, not committed.
- Hover glosses are dictionary lookups on segmented words, so they're occasionally context-off (e.g. 早点 "breakfast" vs. "a bit earlier"); the pinyin is always correct.
- Free HF Spaces can't host a 7B — the demo is local. A public Space would need a paid GPU.
